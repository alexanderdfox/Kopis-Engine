# Kopis Engine Scripts

This directory contains utility scripts for building and running the Kopis Engine.

## Available Scripts

- **open-xcode.sh** - Opens the Swift package in Xcode
- **generate-xcodeproj.sh** - Generates .xcodeproj file using xcodegen
- **create-xcodeproj.sh** - Alternative method to create Xcode project
- **fix-xcode.sh** - Troubleshooting script for Xcode compilation issues
- **run-gui.sh** - Launches the GUI application

## Usage

Most scripts are called automatically by the Makefile. You can also run them directly:

```bash
./scripts/run-gui.sh
./scripts/generate-xcodeproj.sh
```
